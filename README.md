# ♟️ Classic Chess | Premium Edition

A modern, high-end web implementation of classic chess. Built with a focus on premium aesthetics, this project features a "frosted glass" (glassmorphism) UI, butter-smooth animations, and a responsive layout that plays perfectly on Desktop, Tablet, and Mobile.

## ✨ Key Features

* **💎 Premium Glassmorphism UI:** Floating boards, blurred modal overlays (`backdrop-filter`), and dynamic shadows create a tangible, 3D feel.
* **📱 Truly Responsive:** * **PC/Tablet (Landscape):** Classic side-by-side layout with wide "graveyard" panels for captured pieces.
  * **Mobile (Portrait):** Automatically stacks vertically, morphing the side panels into horizontal trays to maximize playable screen space.
* **🤖 Built-in AI:** Play against a local friend or challenge the CPU with three distinct difficulty levels (Easy, Medium, Hard) powered by a custom Minimax algorithm.
* **🎭 Cinematic Typography:** Uses the elegant *Cinzel* font for a dramatic, high-stakes aesthetic.
* **⚡ Zero-Lag Graphics:** Pieces use optimized `text-shadow` rendering to ensure smooth scaling and movement without graphical clipping or lag.

---

## 🎮 How to Play

1. **Launch the Game:** Open the `index.html` file in any modern web browser.
2. **Select Mode:** Click **Start Game** and choose your mode from the blurred modal menu:
   * **Local 2-Player:** Pass and play with a friend on the same device.
   * **vs CPU:** Test your skills against the computer (Easy, Medium, or Hard).
3. **Make a Move:** Click your piece to see valid moves (highlighted with circles), then click the target square to execute the move.
4. **Win:** Checkmate the opponent's King to trigger the victory screen!

## 🛠️ Tech Stack

This project is built entirely without heavy frameworks to ensure maximum performance and easiest deployment:
* **HTML5** for semantic structure.
* **CSS3** for Grid/Flexbox layouts, clamp() typography, CSS variables, and complex backdrop filters.
* **Vanilla JavaScript (ES6)** for the chess engine, move validation, DOM manipulation, and AI logic.
* **Lucide Icons** via CDN for crisp, scalable UI iconography.

---

## 🚀 Installation & Setup

Because this is a pure front-end application, no build tools, Node.js, or servers are required.

1. Clone or download this repository.
2. Double-click `index.html` to open it in Chrome, Safari, Firefox, or Edge.
3. Play!
